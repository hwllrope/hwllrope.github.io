-跳繩健身企劃-


在四個月的時間內，每個月會安排不同的動作來進行！

1. 一跳一迴旋，每組跳三分鐘，休息一分鐘，共五組、20分鐘

2. 跑步跳，每組跳三分鐘，休息一分鐘，共五組、20分鐘

3. 一跳二迴旋，每組跳二分鐘，休息二分鐘，共五組、20分鐘

4. 繩操，形式不限、20分鐘(關於繩操的範例可以去第三個分頁看看！)

每週三至四次，健身二十分鐘加十分鐘練習接下來的健身項目。

並由較簡單的動作循序漸進，使每位使用者都能輕鬆駕馭！
