跳繩基本名詞介紹

原地跳 : 原地併腳跳
移動跳 : 跳躍時邊移動位置
一跳一迴旋 : 跳一下，繩子過腳一次
一跳二迴旋 : 跳一下，繩子過腳兩次
一跳三迴旋 : 跳一下，繩子過腳三次
兩跳一迴旋 : 跳兩下，繩子過腳一次
前迴旋 : 繩子由身後經頭頂再到身前並經過腳下進行迴旋
後迴旋 : 繩子由身前經頭頂再到身後並經過腳下進行迴旋
半迴旋 : 繩子只經過腳下未經過頭頂
體側迴旋 : 繩子經過身體左右再經過頭頂最後經過腳下進行迴旋
水平迴旋 : 繩子進行迴旋時要與地面平行
左右前後開合跳 : 一跳併腳，一跳向前(或左右)開

前舉後振 : 一跳右(左)腳向前舉，接著換左(右)腳向後抬
跑步跳 : 繩子落地時，只有單腳著地，做出像跑步的樣子
軸心迴旋 : 以一點為軸心，然後旋轉繩子跳躍一圈
蹲跳 : 雙腳併攏屈膝進行迴旋
轉身迴旋 : 轉換前後迴旋時的轉身
空迴旋 : 繩子不經過頭頂和腳下
交叉 : 雙臂交叉於體前，而不分開
開叉 : 雙臂分開迴旋過腳一次後，再體前交叉迴旋過腳之跳躍動作
單腳跳 : 繩子落地時，只有單腳著地，另一腳不落地並連續跳躍
左右橫擺 : 一跳右腳向右擺，接著換左腳向左擺
點跳 : 跳躍時一腳伸直，另一腳向前(後)點地
踏踢 : 一跳右(左)腳向前抬，一跳右(左)腳向前舉
